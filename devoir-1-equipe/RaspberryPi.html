Le saviez vous? Utilisation du Raspberry Pi

Si votre internet n'atteint pas complètement votre résidence, Le raspberry Pi peut etre utilisé comme un extension de portée wi-fi pour un bas prix en comparaison avec un extendeur de wi-fi dans le marché. Seulement quelques matériels seront nécessaire:

  • Raspberry Pi 2 ou Raspberry Pi 3
  • Carte Micro SD ou Carte SD
  • Source de courant
  • Clé wifi (Le Raspberry Pi contient une clé wifi intégré)

Pour plus d'information avec l'utilisation du Raspberry Pi, regarder ici.
